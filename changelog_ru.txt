0.10.0b [2020-01-17]
+ Доработан дизайн меню.
+ Добавлен кулдаун в 1 секунду для отправки пакетов - операций в меню группы.
+ Убрана система голосования (будет доработана позже). Лидер группы теперь способен кикнуть любого игрока.
+ Добавлен канал чата группы. При наличии группы можно отправлять сообщения в персональный чат группы: /group <message>.
+ Обновлены параметры конфигурации:
 Клиент 
 - "enable_group_menu_key" [boolean] - разрешить добавление клавиши "открыть меню группы"
 
 Сервер 
 - "group_invite_request_expire_time_seconds" [int] - время активности приглашения в группу в секундах
 - "max_players_per_squad" [int](синхронизируется) - количество участников группы в режиме "отряд"
 - "max_players_per_raid" [int](синхронизируется) - количество участников группы в режиме "рейд"
 - "max_players_per_party" [int](синхронизируется) - количество участников группы в режиме "компания"
 - "enable_group_chat" [boolean](синхронизируется) - разрешить использование персонального чата группы

+ Добавлены команды:
 Сервер [/oxygens]
 - Запросить синхронизацию данных группы: /oxygens groups -sync-group
 
 Клиент [/oxygenc]
 - Открыть меню группы: /oxygenc groups
 - Сбросить данные клиента: /oxygenc groups -reset-data

+ Добавлены привилегии:
 - "group:allowGroupCreation" [id - 200, type - BOOLEAN] - возможность создать группу. По умолчанию любой игрок может создавать группы.

+ Время инициализации группы увеличено до 5 секунд.
+ Доработаны метки участников группы.
+ Добавлено отображение поглощения, запаса брони и активных эффектов для участников в оверлэе группы.
+ Добавлены клиентские настройки:
 Общие
 - возможность открыть меню группы через общее меню (Oxygen Menu)
 - скрыть оверлэй группы
 - скрыть отображение активных эффектов в оверлэе для участников группы
 
 Интерфейс
 - расположение меню
 - горизонтальное смещение оверлэя группы
 - вертикальное смещение оверлэя группы
 - цвет фона индикатора состояния здоровья
 - цвет индикатора состояния здоровья участника оффлайн
 - цвет индикатора состояния здоровья
 - цвет индикатора поглощния
 - цвет индикатора брони
 - формат отображения состояния здоровья
